# Public Second Brain Example

This folder shows the public-safe structure of an Obsidian-compatible knowledge graph. It is not a full private vault.

Open `Home.md` first.

The example mirrors the structure used by the broader stack:

- `proyectos/` — active projects;
- `conceptos/` — durable concepts;
- `entidades/` — agents, organizations, or relevant actors;
- `inbox/` — temporary notes before triage;
- `attachments/` — images or supporting files.
