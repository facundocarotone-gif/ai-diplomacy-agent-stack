# Example: Morning AI Governance Brief

This is a sanitized example structure.

## Sections

1. Top developments
2. UN / multilateral relevance
3. Technical AI developments
4. Policy implications
5. Items requiring follow-up
6. Sources to read

## Example prompt pattern

```txt
Produce a concise AI governance briefing for a policy professional. Prioritize UN, multilateral, regional governance, and major technical AI developments. Avoid repeating yesterday's items unless there is a new development. Separate facts from analysis. Flag uncertainty.
```
